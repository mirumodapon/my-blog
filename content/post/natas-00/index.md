## 關於 overthewire

OverTheWire 是個社群，提供了一系列實作導向 WarGames 這個網站，幫助學習者循序漸進地掌握電腦系統、Linux、網路、密碼學、逆向工程以及 Web Security 等資安領域知識。

OverTheWire 提供多種不同主題的 WarGames，例如：

- Bandit：Linux 指令、檔案系統與基礎資安入門
- Natas：Web Security 與伺服器端漏洞
- Leviathan：權限提升與系統安全
- Krypton：密碼學基礎與古典密碼破解
- Narnia：二進位漏洞與程式安全
- Behemoth：進階權限提升與漏洞利用
- Utumno：Linux 系統安全與 Buffer Overflow
- Maze：綜合型高難度挑戰
